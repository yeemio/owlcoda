CLI commands

Every owlcoda subcommand and flag, grouped by what you actually use them for.

Run owlcoda --help to see this list inline. The grouping below mirrors --help output.

Daily

owlcoda                          # native interactive REPL (default)
owlcoda -p "list all .ts files"  # headless one-shot
owlcoda --resume last            # resume the most recent session
owlcoda --model <alias>          # pick a model by id, alias, or partial match

Setup & diagnostics

owlcoda init                     # create config.json (auto-detects local backend)
owlcoda doctor                   # environment + backend health check
owlcoda config                   # show active config + resolved models
owlcoda models                   # list configured models + reachability
owlcoda validate                 # config schema + semantic checks
owlcoda health                   # proxy + runtime + per-model health
owlcoda inspect                  # recent request/response exchanges
owlcoda audit                    # request audit log (--errors, --slow N)
owlcoda cache                    # response cache stats
owlcoda logs                     # show recent log entries
owlcoda export [--json|--env]    # sanitized config dump for sharing
owlcoda benchmark                # latency / throughput per model
owlcoda completions <shell>      # bash / zsh / fish completion script

Daemon & live clients

owlcoda start                    # background proxy daemon
owlcoda stop [--force]
owlcoda status                   # daemon + live client registry
owlcoda server                   # foreground proxy
owlcoda serve                    # standalone API server (preflight + health)
owlcoda clients                  # list active live REPL clients
owlcoda clients detach <id>      # detach without stopping the daemon

Browser admin

owlcoda ui                       # print one-shot admin URL
owlcoda ui --open-browser        # launch in default browser
owlcoda admin                    # alias of `ui`
owlcoda ui --route <r> --select <id>  # focused handoff

Sessions & skills

owlcoda sessions                 # --limit N, --tag T, --json
owlcoda skills info|list|show|synth|delete|export|import
owlcoda skills stats|cleanup|search|match

Training data (opt-in)

owlcoda training status          # collection statistics
owlcoda training scan            # batch-score historical sessions
owlcoda training report          # quality distribution
owlcoda training export jsonl|sharegpt|insights
owlcoda training clear
owlcoda training path            # print collected.jsonl path

Inside the REPL, /help lists slash commands. See Native REPL for the most useful ones.
