安装

在 macOS / Linux / Windows-WSL 上 5 分钟内跑通 OwlCoda。

OwlCoda 当前以源码形式分发。npm / Homebrew / 独立二进制将在 1.0 之后提供。

前置条件

  • Node.js ≥ 18（推荐 Node 20+）
  • 本地 OpenAI 兼容推理后端（Ollama / LM Studio / vLLM / 自建 router） —— 纯本地运行必需
  • macOS / Linux / Windows（Windows 推荐 WSL）

步骤

git clone https://github.com/yeemio/owlcoda.git
cd owlcoda
npm install
npm run build
npm link             # 之后 `owlcoda` 全局可用

如果 npm link 失败

通常是全局 npm prefix 没写权限。

  • 方案 A — sudo： sudo npm link
  • 方案 B — 用户级 prefix： 
    npm config set prefix ~/.local
export PATH=~/.local/bin:$PATH    # 加进你的 shell profile
cd /path/to/owlcoda && npm link
  • 方案 C — 完全不做 link： node /path/to/owlcoda/dist/cli.js …

验证安装

owlcoda --version
owlcoda doctor

owlcoda doctor 会检查 Node 版本、本地 runtime 可达性、已配置的模型。如果某项 fail，看 排错。
