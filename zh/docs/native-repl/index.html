Native REPL

Slash 命令、工具、会话、技能、训练管线 —— OwlCoda 内你做的每件事。

跑 owlcoda 进入 native REPL。它看起来像终端，因为它本来就是终端。

常用 slash 命令

命令作用
/help列出全部 slash 命令，按职能分组
/model打开模型 picker；切换当前 model
/cost当前会话 token + 成本明细
/budget已用 vs 估算上下文窗口
/perf各 model 延迟 + tokens/sec + 成功率
/doctor最近错误、运行时长、trace 状态
/config当前 model / proxy / router / flag / 会话信息
/trace切换 JSON 请求 / 响应 tracing
/tokens累计 input / output / total token
/sessions列最近会话；恢复任意一次
/tag <name>给当前会话打标签
/branch在当前消息处 fork 当前会话
/history应用内回滚（PgUp / PgDn / Ctrl+↓ 也行）
/skills当前请求被注入的 skill + 匹配分
/dashboard紧凑状态总览

常用工具

每个工具由模型在需要时调用 —— 你不会直接调。

工具作用
Bash跑 shell 命令（破坏性操作有 prompt）
Read按绝对路径读文件
Write写或覆盖文件

还有 30+ 个 —— REPL 内 /tools 看全部。

会话

每次对话自动落到 ~/.owlcoda/sessions/<id>.json。恢复：

owlcoda --resume last           # 上次会话
owlcoda --resume <id>           # 指定 id（从 owlcoda sessions 拿）

可给会话打标签（/tag launch-blocker）、按标签搜（owlcoda sessions --tag launch-blocker）、从任意消息处分支（/branch）。

技能 (L2)

OwlCoda 把用户输入与精选方法论 pack + 用户安装的 ~/.owlcoda/skills/ 做匹配，命中的 skill 在请求时注入 system prompt。可调：

{
  "skillInjection": true,
  "skillTopK": 2,
  "skillThreshold": 0.1
}

跑 owlcoda skills info 看当前加载了什么。

训练管线 (L3，opt-in / 默认关闭)

{ "trainingCollection": false }

或运行时覆盖：

OWLCODA_TRAINING_COLLECTION=1 owlcoda

开启时，会话按 5 维度加权打分、PII 脱敏、追加到 ~/.owlcoda/training/collected.jsonl。导出：

owlcoda training export jsonl --min-quality 70
owlcoda training export sharegpt --tool-calls-only
