CLI 命令

owlcoda 全部子命令与参数，按你实际会用到的场景分组。

跑 owlcoda --help 可以看到这份命令清单。下面的分组与 --help 输出一致。

日常

owlcoda                          # 默认进入 native 交互式 REPL
owlcoda -p "list all .ts files"  # 非交互模式 (headless one-shot)
owlcoda --resume last            # 恢复上次会话
owlcoda --model <alias>          # 按 id / 别名 / 部分匹配选模型

配置 & 诊断

owlcoda init                     # 创建 config.json（自动探测本地 backend）
owlcoda doctor                   # 环境与 backend 健康检查
owlcoda config                   # 查看当前配置 + 已解析模型
owlcoda models                   # 列出已配置模型 + 可达性
owlcoda validate                 # config schema + 语义检查
owlcoda health                   # proxy + runtime + per-model 健康
owlcoda inspect                  # 最近请求 / 响应记录
owlcoda audit                    # 请求 audit log（--errors / --slow N）
owlcoda cache                    # 响应缓存统计
owlcoda logs                     # 最近日志
owlcoda export [--json|--env]    # 脱敏 config 导出（分享用）
owlcoda benchmark                # 各 model 延迟 / 吞吐
owlcoda completions <shell>      # bash / zsh / fish 自动补全脚本

Daemon & 活跃 client

owlcoda start                    # 后台 proxy daemon
owlcoda stop [--force]
owlcoda status                   # daemon + 活跃 client 注册表
owlcoda server                   # 前台 proxy
owlcoda serve                    # 独立 API server (preflight + health)
owlcoda clients                  # 列活跃 live REPL client
owlcoda clients detach <id>      # 不停 daemon 解绑某 client

Browser admin

owlcoda ui                       # 打印 one-shot admin URL
owlcoda ui --open-browser        # 直接在浏览器打开
owlcoda admin                    # ui 的别名
owlcoda ui --route <r> --select <id>  # 焦点 handoff

会话 & 技能

owlcoda sessions                 # --limit N / --tag T / --json
owlcoda skills info|list|show|synth|delete|export|import
owlcoda skills stats|cleanup|search|match

训练数据（opt-in）

owlcoda training status          # 收集统计
owlcoda training scan            # 批量给历史会话打分
owlcoda training report          # 质量分布
owlcoda training export jsonl|sharegpt|insights
owlcoda training clear
owlcoda training path            # 打印 collected.jsonl 路径

REPL 内 /help 列 slash 命令。常用的见 Native REPL。
